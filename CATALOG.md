## 维护约定

- Markdown 文档优先使用 UTF-8 编码。
- 日期类文件建议统一为 `YYYY-MM-DD.md`，历史文件可以保留原命名。
- 含图片的笔记保持图片与 Markdown 同目录，使用相对路径引用。
- 内容索引只描述文档用途和主题，不直接覆盖原始记录。
