| 目录 | 内容 | 入口 |
|---|---|---|
| `Food/` | 美食月餐厅计划，包含地点、人数、预约方式、价格和计划时间。 | [Food/README.md](Food/README.md) |
| `IDP/` | Individual Development Plan，职业发展目标、能力现状和行动计划。 | [IDP/README.md](IDP/README.md) |
| `Interview/` | 面试材料、业务知识、技术知识、知识复习和截图材料。 | [Interview/README.md](Interview/README.md) |

更完整的文件级索引见 [CATALOG.md](CATALOG.md)。

## 快速入口

- 查找某篇历史文档：优先打开 [CATALOG.md](CATALOG.md)。
- 新增或整理文档：参考 [MAINTENANCE.md](MAINTENANCE.md)。
- 查看全局审计：打开 [docs/content-audit.md](docs/content-audit.md)。
- 准备面试复习：从 [Interview/template.md](Interview/template.md) 或 [Interview/README.md](Interview/README.md) 开始。
- 写周报：复制 [WeeklyReport/周报模板.md](WeeklyReport/周报模板.md)。
- 查看职业发展材料：进入 [IDP/README.md](IDP/README.md)。

## 推荐使用方式

1. 新增周报时，复制 [WeeklyReport/周报模板.md](WeeklyReport/周报模板.md)，按日期命名为 `YYYY-M-D.md` 或统一为 `YYYY-MM-DD.md`。
2. 新增面试笔记时，优先放在 `Interview/social/YYYYMMDD/`，Markdown 文件使用同一天的 `YYYY-MM-DD.md` 命名；截图保存在同目录，方便相对路径引用。
3. 新增招聘、IDP、生活计划等独立主题时，放入对应一级目录，并在该目录的 `README.md` 和根目录 [CATALOG.md](CATALOG.md) 中补一行索引。
4. 保留历史原文，不在整理索引时改写旧笔记内容；需要复盘时可新建总结文档，避免覆盖原始记录。
